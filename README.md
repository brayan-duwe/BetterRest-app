# BetterRest 😴

*This app was built using SwiftUI.*

This is an app where the user enters their wake-up time, desired hours of sleep, and how many cups of coffee they consumed during the day.

The app returns the ideal bedtime based on a machine learning model (Core ML), trained on thousands of data points related to sleep.

## Project Files

The app structure is very simple, containing the ContentView and the SleepCalculator (ML Model).
- [ContentView.swift](/BetterRest/ContentView.swift)
- [SleepCalculator.mlmodel](/BetterRest/SleepCalculator.mlmodel)
